Getting Started

  1. Install the native host: cd ~/Developer/nooma-stack/pkrelay && ./native-host/install.sh
  2. Set your browser name above (e.g., "Arc")
  3. Click Test Connection to verify the native messaging host
  4. Click the PKRelay icon in the toolbar to manage tab permissions
  5. Attach tabs by clicking the green/gray dot next to each tab in the popup
  6. Add PKRelay as an MCP server in your AI tool's config
