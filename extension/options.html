Getting Started

  1. Install the CDP bridge: cd ~/pkrelay && ./install.sh
  2. Set your browser name above (e.g., "Arc")
  3. Click Test CDP Server to verify: http://127.0.0.1:18792/json/version
  4. Click the PKRelay icon in the toolbar to manage tab permissions
  5. Attach tabs by clicking the green/gray dot next to each tab in the popup
  6. OpenClaw config should have: "cdpUrl": "http://127.0.0.1:18792"
